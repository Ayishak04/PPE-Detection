PPE Safety Kit Object Detection

Overview

This project aims to detect PPE safety kits (Personal Protective Equipment) using the YOLO (You Only Look Once) object detection algorithm. The model detects various types of equipment including 'Hardhat', 'Mask', 'No-Hardhat', 'No-Mask', 'No-Safety Vest', 'Person', 'Safety Cone', 'Safety Vest', 'Machinery', and 'Vehicle'. The target audience includes organizations or individuals aiming to enhance workplace safety in hazardous environments.

Dataset

The dataset used for training is the PPE Detection dataset available on Roboflow. It includes classes such as 'Hardhat', 'Mask', and others, with 2.6k images for training, 114 for validation, and 82 for testing. Each image has dimensions of 640x640.
